GTPrint Printing Scripts

The GTPrint Printing Scripts are a set of shell scripts and AppleScripts that allow you print to the printers in the Georgia Tech library, student center, and Central Printing service from the Mac OS X print dialog. The GTPrint Scripts consist of about 500 lines of code.

Working on this project taught me a lot about:
  • how Mac OS X's PDF Services mechanism works,
  • how to write advanced AppleScripts, and
  • how to build a Mac OS X installer.
