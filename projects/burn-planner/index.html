Burn Planner

Burn Planner helps you burn a series of files to multiple DVDs quickly by automatically allocating files among DVDs and naming the DVDs.

Download for Mac OS X 10.5 and later

History

I wrote this program because I archive a lot of files to DVDs and I needed a program to make the process faster.
